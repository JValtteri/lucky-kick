# "Lucky-Kick" Frisbee-Golf
This will be a top down frisbee-golf game made with PyGame and Python3

### PRE-ALPHA ###

## Features
- Frisbee turn
- Frisbee fade
- hyzer / anhyzer
- kicks from trees
- Throw power
